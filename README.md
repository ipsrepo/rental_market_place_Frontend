# Rental Marketplace - Frontend

This is the frontend repository for the Rental Marketplace application. It provides a user interface for browsing, listing, and managing rental items.

## 🚀 Tech Stack

- **React 19** - UI library
- **Vite 8** - Build tool and development server
- **Tailwind CSS 4** - Utility-first CSS framework
- **React Router 7** - Navigation and routing
- **Redux Toolkit 2** - State management
- **Axios** - HTTP client for API requests


## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/rental_market_place_Frontend.git

# Navigate to project directory
cd rental_market_place_Frontend

# Install dependencies
npm install

# Start development server
npm run dev
```
